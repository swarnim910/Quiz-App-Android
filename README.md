# Android Quiz App

An Android quiz application built using Kotlin with Firebase Authentication and PHP-MySQL backend.

## Features
- User Registration & Login (Firebase Auth)
- Dynamic quiz questions from PHP API
- Score calculation and result screen
- Clean UI with multiple activities

## Tech Stack
- Kotlin, XML
- Firebase Authentication
- PHP, MySQL (XAMPP)
- Retrofit for API calls

## Setup
1. Clone repo
2. Open in Android Studio
3. Add your own `google-services.json`
4. Update API base URL
5. Run on emulator/device
